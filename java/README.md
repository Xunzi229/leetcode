# 没事刷刷题


```
两数之和（Two Sum） - 数组，哈希表
两个有序数组的中位数（Median of Two Sorted Arrays） - 二分查找
最长无重复字符的子串（Longest Substring Without Repeating Characters） - 字符串，滑动窗口
寻找两个正序数组的中位数（Median of Two Sorted Arrays） - 二分查找
最长回文子串（Longest Palindromic Substring） - 字符串，动态规划
无重复字符的最长子串（Longest Substring Without Repeating Characters） - 哈希表，滑动窗口
盛最多水的容器（Container With Most Water） - 数组，双指针
三数之和（3Sum） - 数组，双指针
最接近的三数之和（3Sum Closest） - 数组，双指针
电话号码的字母组合（Letter Combinations of a Phone Number） - 字符串，回溯
删除链表的倒数第N个节点（Remove Nth Node From End of List） - 链表，双指针
括号生成（Generate Parentheses） - 字符串，回溯
合并K个升序链表（Merge k Sorted Lists） - 链表，分治，优先队列
最长公共前缀（Longest Common Prefix） - 字符串
有效的括号（Valid Parentheses） - 栈
合并两个有序链表（Merge Two Sorted Lists） - 链表
二叉树的层序遍历（Binary Tree Level Order Traversal） - 树，广度优先搜索
二叉树的最大深度（Maximum Depth of Binary Tree） - 树，深度优先搜索
二叉树的中序遍历（Binary Tree Inorder Traversal） - 树，深度优先搜索
岛屿数量（Number of Islands） - 深度优先搜索，广度优先搜索，并查集
```
